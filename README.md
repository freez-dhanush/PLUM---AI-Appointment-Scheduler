# PLUM--AI-Appointment-Scheduler

A simple Node.js backend API built using **Express**, supporting **image uploads** with Multer and tested using **Postman** and **VS Code Thunder Client**.

## Features

- REST API using **Express.js**
- Image/file upload handling using **Multer**
- Environment variable support using **dotenv**
- Tested using **Postman** and **Thunder Client**
- Clean and simple folder structure

## Technologies Used

- Node.js  
- Express.js  
- Multer (for image uploads)  
- dotenv  
- Postman / Thunder Client  
- VS Code
